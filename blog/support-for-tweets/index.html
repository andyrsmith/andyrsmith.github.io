Support for Tweets

You can easily embed Tweets in your notes. Simply use the special Markdown code:

[twitter https://twitter.com/collected_notes/status/1271378401148821504]

> Note We send the Do Not Track parameter to Twitter automatically.

The Markdown code above will render:

[twitter https://twitter.com/collected_notes/status/1271378401148821504]
